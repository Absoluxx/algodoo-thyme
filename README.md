## algodoo-thyme README

### Description

This extension aims to provide language support to the proprietary scripting language of Algodoo, Thyme.

### Features

This extension is currently a work in progress. The current features are only color highlighting.

### How to add the extension

In the repository, locate the file formatted as "algodoo-thyme-x.x.x.vsix" and download that file. Then, open VSCode, go into the top search bar, and paste in ">Extensions: Install from VSIX...". Locate the downloaded file in your computer, confirm it, and the extension should be configured. Create a new file with the extension .thy to have the features take effect on your code.
