# algodoo-thyme README

## Description

This extension aims to provide language support to the proprietary scripting language of Algodoo, Thyme.

## Features

This extension is currently a work in progress. The current features are only color highlighting.

## Release Notes

### 0.1.0

- Coloring for data types (numbers, strings, bools)
- Coloring for the print() function.
- Coloring for the e and Scene objects + their child properties.
